# Changelog

All notable changes to Screenshot Beautifier will be documented in this file.

## [1.0.0] - 2026-01-02

### Added
- Initial release
- 10 gradient background presets
- 10 solid color options
- 4 shadow levels (none, soft, medium, hard)
- Adjustable padding (16-128px)
- Adjustable border radius (0-32px)
- Paste from clipboard
- Drag and drop support
- File upload
- Copy to clipboard
- Download as PNG
- Modern dark UI
